The Three Autonomous Agents

The Archivist

Monitors your local directory structure, auto-registers files, processes frontmatter metadata, and indexes file changes.

  • Directory auto-sync watch
  • SQLite metadata tracking
  • Category structure management

The Researcher

Generates sentence-level embeddings, builds the vector database, and executes hybrid search algorithms for RAG answering.

  • ChromaDB local vector store
  • Full-text & vector hybrid search
  • Citations & insights rendering

The Task Master

Extracts scheduled tasks, deadlines, and markdown checklists to prepare dynamic briefs and track notes to-do progress.

  • Automatic checklist extraction
  • Daily brief compilations
  • Status badge lifecycle tracking
